# GPT e OpenAI

## Introdução

A OpenAI é uma empresa líder em pesquisa de IA, criadora dos modelos GPT (Generative Pre-trained Transformer) que revolucionaram o campo de Large Language Models.

## Modelos Principais

### GPT-4 Turbo
- 128k tokens de contexto
- Multimodal (texto + visão)
- Lançamento: 2023
- Mais rápido e econômico que GPT-4

### GPT-4
- Modelo flagship da OpenAI
- Capacidades avançadas de raciocínio
- Multimodal

### GPT-3.5 Turbo
- Base do ChatGPT gratuito
- Rápido e econômico
- 16k contexto

## Aplicações
- ChatGPT
- GitHub Copilot (via Codex)
- Inúmeras integrações via API
