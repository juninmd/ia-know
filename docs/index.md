## Por que IA Know?

Este repositório foi criado para ser uma **base de conhecimento centralizada** para desenvolvedores que desejam entender e utilizar:

- **LLMs (Large Language Models)**: Modelos de linguagem de grande escala
- **Agentes de IA**: Sistemas autônomos que executam tarefas
- **Ferramentas de Desenvolvimento**: Claude Code, GitHub Copilot, Cursor, etc.
- **AGI**: Conceitos de Inteligência Artificial Geral
- **Boas Práticas**: Como usar IA de forma eficiente e responsável
