# Cursor IDE

## Visão Geral
Cursor é uma IDE completa (fork do VS Code) otimizada para desenvolvimento com IA integrada.

## Características Principais

### Composer Mode
- Edição multi-arquivo com IA
- Compreensão de projeto completo
- Aplicação automática de mudanças

### Chat Contextual
- Conversa sobre código
- Entende toda a codebase
- Geração de código inline

### Ctrl+K
- Edição inline rápida
- Aceitar/rejeitar mudanças
- Iteração rápida

## Vantagens
- IDE completa (não apenas plugin)
- Contexto profundo do projeto
- Interface otimizada para IA

## Preços
- Gratuito com limitações
- Pro: $20/mês
