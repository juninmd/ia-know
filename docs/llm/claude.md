# Claude (Anthropic)

## Visão Geral
Anthropic desenvolveu a família Claude de LLMs com foco em segurança e conversação útil.

## Modelos
- **Claude 3.5 Sonnet**: Melhor balanço performance/custo
- **Claude 3 Opus**: Mais capaz
- **Claude 3 Haiku**: Mais rápido e econômico

## Diferenciais
- Janela de contexto até 200k tokens
- Constitutional AI para segurança
- Excelente para análise longa e coding
