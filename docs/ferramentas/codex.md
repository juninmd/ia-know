# OpenAI Codex

## Visão Geral
Codex é uma versão do GPT-3 especializada em código, treinada em bilhões de linhas de código público.

## Base do GitHub Copilot
Codex é o motor por trás do GitHub Copilot.

## Capacidades
- Geração de código em 100+ linguagens
- Tradução entre linguagens
- Explicação de código
- Debugging

## Status
A API pública do Codex foi descontinuada, mas a tecnologia continua no Copilot.
