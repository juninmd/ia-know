# Glossário - IA Know

Glossário completo de termos relacionados a Inteligência Artificial, Large Language Models, Agentes e desenvolvimento com IA.

---

## A

### AGI (Artificial General Intelligence)
Inteligência Artificial Geral - sistemas de IA com capacidades cognitivas equivalentes ou superiores aos humanos em qualquer tarefa intelectual.

### Agent (Agente)
Sistema de IA capaz de realizar tarefas de forma autônoma, percebendo o ambiente, tomando decisões e executando ações para atingir objetivos específicos.

### AI Alignment (Alinhamento de IA)
Processo de garantir que sistemas de IA operem de acordo com valores e intenções humanas.

### ANI (Artificial Narrow Intelligence)
Inteligência Artificial Restrita - sistemas especializados em tarefas específicas (estado atual da maioria das IAs).

### Anthropic
Empresa de pesquisa em IA focada em segurança, criadora do Claude.

### API (Application Programming Interface)
Interface que permite que aplicações se comuniquem com modelos de IA via requisições programáticas.

### ASI (Artificial Super Intelligence)
Inteligência Artificial Superior - hipotética IA que supera significativamente a inteligência humana em todos os aspectos.

### Attention Mechanism (Mecanismo de Atenção)
Técnica que permite modelos focarem em partes relevantes da entrada ao processar informações.

### AutoGPT
Agente autônomo que usa GPT-4 para executar tarefas complexas de forma independente.

---

## B

### BabyAGI
Framework de agente autônomo que usa LLMs para criar, priorizar e executar tarefas.

### Backpropagation
Algoritmo fundamental para treinar redes neurais, calculando gradientes para ajustar pesos.

### BERT (Bidirectional Encoder Representations from Transformers)
Modelo de linguagem do Google que lê texto bidirecionalmente.

### Bias (Viés)
Tendências sistemáticas em dados de treinamento ou outputs de modelos que podem levar a resultados injustos ou imprecisos.

### BLOOM
Large Language Model multilíngue open-source desenvolvido pela BigScience.

---

## C

### Chain-of-Thought (CoT)
Técnica de prompting que solicita ao modelo que explique seu raciocínio passo a passo.

### ChatGPT
Interface conversacional da OpenAI baseada em GPT-3.5/GPT-4.

### Claude
Família de LLMs desenvolvida pela Anthropic, focada em segurança e conversação útil.

### Claude Code
Ferramenta CLI da Anthropic para desenvolvimento assistido por IA.

### Codex
Modelo da OpenAI especializado em código, base do GitHub Copilot.

### Context Window (Janela de Contexto)
Quantidade máxima de tokens que um modelo pode processar de uma vez (ex: 4k, 128k, 1M tokens).

### Constitutional AI
Método da Anthropic para treinar IAs usando princípios éticos definidos.

### Copilot
Assistente de código da GitHub/Microsoft baseado em OpenAI Codex.

### CrewAI
Framework para orquestrar múltiplos agentes de IA especializados trabalhando juntos.

### Cursor
IDE (fork do VS Code) otimizada para desenvolvimento com IA integrada.

---

## D

### DALL-E
Modelo da OpenAI que gera imagens a partir de descrições textuais.

### Devin AI
Agente de IA anunciado como "primeiro engenheiro de software autônomo".

### Diffusion Models
Classe de modelos generativos usados para criar imagens (ex: Stable Diffusion).

### DSPy
Framework para programação declarativa com LLMs.

---

## E

### Embedding
Representação vetorial de texto, código ou outros dados em espaço multidimensional.

### Emergent Abilities
Capacidades que aparecem em modelos grandes mas não em modelos menores.

### Epoch
Uma passagem completa através de todo o dataset de treinamento.

### Ethics (Ética em IA)
Considerações morais e sociais no desenvolvimento e uso de IA.

---

## F

### Few-Shot Learning
Capacidade de aprender tarefas com poucos exemplos fornecidos no prompt.

### Fine-Tuning
Processo de treinar um modelo pré-treinado em dados específicos para especialização.

### Foundational Models
Modelos grandes treinados em dados diversos que servem de base para múltiplas aplicações.

### Function Calling
Capacidade de LLMs de invocar funções/ferramentas externas estruturadamente.

---

## G

### Gemini
Família de modelos multimodais do Google (sucessor do Bard/PaLM).

### Generative AI
IA capaz de criar conteúdo novo (texto, imagem, áudio, vídeo, código).

### GitHub Copilot
Assistente de programação que sugere código em tempo real.

### GPT (Generative Pre-trained Transformer)
Arquitetura de modelo de linguagem da OpenAI.

### Gradient Descent
Algoritmo de otimização usado para treinar modelos de machine learning.

### Grounding
Conectar LLMs a fontes de informação factuais para reduzir alucinações.

### Guardrails
Regras e limitações impostas a sistemas de IA para garantir outputs seguros e apropriados.

---

## H

### Hallucination (Alucinação)
Quando um modelo gera informações falsas ou sem base factual de forma convincente.

### HuggingFace
Plataforma e empresa focada em democratizar IA, oferecendo modelos e datasets open-source.

### Hyperparameters
Parâmetros de configuração do treinamento (learning rate, batch size, etc).

---

## I

### In-Context Learning
Aprendizado através de exemplos fornecidos no prompt sem modificar os pesos do modelo.

### Inference
Processo de usar um modelo treinado para fazer previsões em novos dados.

### Instruction Tuning
Fine-tuning de modelos para seguir instruções humanas melhor.

### Instructor
Biblioteca para obter outputs type-safe e estruturados de LLMs.

---

## J

### JSON Mode
Modo de output onde LLMs retornam dados estruturados em formato JSON.

---

## K

### Knowledge Base
Base de dados estruturada usada para RAG ou fine-tuning.

### Knowledge Cutoff
Data limite do conhecimento de um modelo baseado em quando foi treinado.

---

## L

### LangChain
Framework popular para construir aplicações com LLMs, incluindo agentes e RAG.

### LangGraph
Extensão do LangChain para controle de fluxo avançado com grafos.

### LLaMA (Large Language Model Meta AI)
Família de modelos open-source da Meta.

### LLM (Large Language Model)
Modelo de linguagem de grande escala treinado em vastas quantidades de texto.

### LlamaIndex
Framework especializado em RAG e indexação de dados para LLMs.

### LoRA (Low-Rank Adaptation)
Técnica eficiente para fine-tuning que modifica apenas uma pequena parte do modelo.

---

## M

### Machine Learning (ML)
Campo da IA focado em sistemas que aprendem de dados.

### MCP (Model Context Protocol)
Protocolo para padronizar comunicação entre aplicações e LLMs.

### MetaGPT
Framework multi-agent que simula uma empresa de software com diferentes "funcionários" de IA.

### Mistral
Empresa europeia criando modelos LLM open-source de alta performance.

### MoE (Mixture of Experts)
Arquitetura onde diferentes "experts" (sub-modelos) são ativados para diferentes inputs.

### Multi-Agent System
Sistema com múltiplos agentes de IA trabalhando juntos, coordenados ou competitivos.

### Multimodal
Modelos capazes de processar múltiplos tipos de dados (texto, imagem, áudio, vídeo).

---

## N

### Neural Network (Rede Neural)
Modelo computacional inspirado no cérebro humano com camadas de neurônios artificiais.

### NLP (Natural Language Processing)
Processamento de Linguagem Natural - campo da IA focado em entender e gerar linguagem humana.

---

## O

### OpenAI
Empresa de pesquisa em IA criadora do GPT, ChatGPT, DALL-E e Codex.

### Open-Source
Modelos ou software com código aberto, permitindo uso, modificação e distribuição.

### Overfitting
Quando um modelo aprende detalhes específicos dos dados de treino ao invés de padrões gerais.

---

## P

### Parameters (Parâmetros)
Pesos ajustáveis em um modelo neural (ex: GPT-4 tem trilhões de parâmetros).

### PaLM (Pathways Language Model)
Modelo de linguagem do Google, predecessor do Gemini.

### Perplexity
Métrica que mede quão bem um modelo prevê texto; também nome de um search engine com IA.

### Prompt
Texto de entrada fornecido a um LLM para gerar uma resposta.

### Prompt Engineering
Arte e ciência de criar prompts eficazes para obter melhores resultados de LLMs.

### Prompt Injection
Ataque onde inputs maliciosos tentam manipular o comportamento do modelo.

---

## Q

### Quantization
Técnica para reduzir o tamanho de modelos convertendo pesos de precisão alta para baixa.

### QLoRA
Combinação de quantização e LoRA para fine-tuning ultra-eficiente.

---

## R

### RAG (Retrieval Augmented Generation)
Técnica que combina LLMs com busca em bases de conhecimento para respostas factuais.

### Reinforcement Learning (RL)
Aprendizado através de recompensas e punições.

### RLHF (Reinforcement Learning from Human Feedback)
Técnica de treinar modelos usando feedback humano sobre outputs.

### Role Prompting
Técnica de definir um papel para o modelo (ex: "Você é um especialista em Python").

---

## S

### Semantic Kernel
Framework da Microsoft para integrar IA em aplicações.

### Semantic Search
Busca baseada em significado ao invés de palavras-chave exatas.

### Sentiment Analysis (Análise de Sentimento)
Determinar emoção ou opinião expressa em texto.

### Stable Diffusion
Modelo open-source de geração de imagens.

### System Prompt
Instruções iniciais que definem comportamento e personalidade do modelo.

---

## T

### Tabnine
Assistente de código com IA que oferece autocompletar.

### Temperature
Parâmetro que controla aleatoriedade/criatividade das respostas (0 = determinístico, 1+ = criativo).

### Token
Unidade básica de texto processada por LLMs (pode ser palavra, parte de palavra ou caractere).

### Tokenization
Processo de dividir texto em tokens.

### Top-k Sampling
Técnica de geração que considera apenas os k tokens mais prováveis.

### Top-p (Nucleus) Sampling
Técnica que considera tokens cuja probabilidade acumulada atinge p.

### Training
Processo de ajustar parâmetros de um modelo usando dados.

### Transfer Learning
Usar conhecimento de uma tarefa para melhorar desempenho em outra.

### Transformer
Arquitetura de rede neural baseada em attention, base dos LLMs modernos.

---

## U

### Underfitting
Quando modelo é muito simples para capturar padrões nos dados.

---

## V

### Vector Database
Banco de dados otimizado para armazenar e buscar embeddings.

### Vision Transformer (ViT)
Aplicação de transformers para processar imagens.

### VitePress
Gerador de sites estáticos otimizado para documentação (usado neste projeto).

---

## W

### Weights (Pesos)
Parâmetros ajustáveis em redes neurais que determinam o comportamento do modelo.

### Whisper
Modelo da OpenAI para reconhecimento de fala (speech-to-text).

---

## Z

### Zero-Shot Learning
Capacidade de realizar tarefas sem exemplos, apenas com instruções.

---
